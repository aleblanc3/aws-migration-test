Horizontal radio button component

This component accepts:

  • label="your.translation.key" to set the label above the buttons
  • name="unique-name" in case you use it more than once on a page
  • [options]="yourArray" with your array of radio button labels, values, and icons
  • [(selected)]="yourSelectedButton" to define which button is currently selected
  • (selectedChange)="yourFunction($event) to define what happens when the buttons are clicked

The component emits the value of the selected radio button.
